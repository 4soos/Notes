## Bond 设备
虚拟网卡bond是通过多张网卡绑定为一个逻辑网卡，实现本地网卡的冗余，带宽扩容和负载均衡，在生产场景中是一种常用的技术。 其工作原理说起来也挺简单，就是虚拟网卡开启混杂模式，接收多张网卡的数据，并改成相同的mac地址。

|                              |                                 |
|       +------------------------------------------------+       |
|       |                    eth2 (one mac)              |       |
|       +------------------------------------------------+       |
|              ↑                                ↑                |
|..............|................................|................|

|        +----------+                     +------------+         |
|              ↑                                 ↑               |
|              |                                 |               |

               ↓                                 ↓
         Physical Network 1              Physical Network 2
